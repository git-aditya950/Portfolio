# Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Modern Dark Theme** - Sleek dark background with lime green (#c8f135) and cyan (#3dffd0) accent colors
- **Custom Cursor** - Unique custom cursor implementation for enhanced user experience
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Smooth scrolling and animated transitions
- **Professional Layout** - Showcase your projects, skills, and experience

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with custom properties and animations
- **JavaScript** - Interactive elements and cursor tracking
- **Google Fonts** - Space Mono and Syne typefaces

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/git-aditya950/Portfolio.git
```

2. Navigate to the directory:
```bash
cd Portfolio
```

3. Open `portfolio.html` in your browser or serve with a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Customization

To customize the portfolio for your use:

1. Update the color scheme by modifying CSS variables in the `:root` selector
2. Edit content sections with your information
3. Add or remove portfolio projects as needed
4. Update social links and contact information

### Color Variables

```css
--bg: #0a0a0a;              /* Main background */
--surface: #111111;         /* Surface elements */
--border: #1e1e1e;          /* Borders */
--accent: #c8f135;          /* Primary accent (lime) */
--accent2: #3dffd0;         /* Secondary accent (cyan) */
--text: #f0f0f0;            /* Text color */
--muted: #666;              /* Muted text */
--card: #141414;            /* Card background */
```
